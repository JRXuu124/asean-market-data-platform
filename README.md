# ASEAN Market Data Platform

A comprehensive data platform for cross-border e-commerce entrepreneurs targeting ASEAN markets.

## Features
- Real-time market data from ASEAN countries
- Product and price intelligence
- Competition analysis
- Market entry recommendations
- Automated reporting

## Tech Stack
- **Backend**: FastAPI (Python)
- **Database**: PostgreSQL, Redis, Elasticsearch
- **Async Tasks**: Celery
- **Frontend**: HTML/CSS/JS with Chart.js
- **Containerization**: Docker & Docker Compose

## Quick Start

### Prerequisites
